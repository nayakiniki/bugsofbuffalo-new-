# Indian Cattle and Buffalo Breeds Dataset
# Source: https://github.com/sundram6452/Image-based-breed-recognition-for-cattle-and-buffaloes-of-India/tree/main/dataset
# Created: 2024
# Classes: 12 Indian breeds

This dataset contains images of 12 indigenous Indian cattle and buffalo breeds for breed recognition using computer vision.

## Breed Classes:
1. Gir
2. Sahiwal
3. Red Sindhi
4. Tharparkar
5. Kankrej
6. Ongole
7. Hariana
8. Kangayam
9. Deoni
10. Amritmahal
11. Hallikar
12. Murrah

## Dataset Structure:
- train/images/ - Training images
- train/labels/ - YOLO format annotations
- valid/images/ - Validation images
- valid/labels/ - YOLO format annotations
- test/images/ - Test images
- test/labels/ - YOLO format annotations

## Annotation Format:
YOLO format: <class_id> <center_x> <center_y> <width> <height>
